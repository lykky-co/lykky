How it works

  • Nodes float and cluster by category using D3 force simulation
  • Each category has its own color (green=funding, blue=research, etc.)
  • Click any node → side panel slides in with details + link
  • Drag nodes to explore, zoom/pan the canvas
  • Category legend at top for filtering
  • Data sourced from the MDX resource page frontmatter at build time
